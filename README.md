- TypeScript
- Gatsby.js
- styled-components
- react-spring

## Setup

Clone the project.

Install all packages:

### `npm i`

To start developing, run:

### `gatsby develop`

Open [http://localhost:8000](http://localhost:8000) to view it in the browser.
